# FundFlow - Soroban Contract Bindings

TypeScript library for interacting with the FundFlow Soroban smart contract via Soroban RPC.

## Contract Details

- **Contract Name**: FundFlow Fundraising
- **Contract ID**: `CCEG5OCIXMVF73CFGI52AOTTPTCIN46QBK2CF7HC7TMSO4YJFU7ZQRPQ`
- **Network**: Test SDF Network (September 2015)

## Installation

```bash
# Add to your project's dependencies
npm install contract
# or
bun add contract
```

Or link locally:
```json
"dependencies": {
  "contract": "file:packages/contract"
}
```

## Usage

```typescript
import { Client, networks } from "contract";

const client = new Client({
  ...networks.testnet,
  rpcUrl: "https://soroban-testnet.stellar.org",
});

// Create a campaign
const tx = await client.create_campaign({
  id: 1,
  owner: "G...",
  goal: 10000000000n, // 1000 XLM in stroops
});

// Sign with Freighter and submit
// See full example in hooks/contract.ts
```

## Updating

To regenerate after contract changes:

```bash
# Rebuild contract
cd ../contract && cargo build --release

# Regenerate bindings

  --network testnet \
```

## Methods

| Method | Description |
|--------|-------------|
| `create_campaign` | Create new fundraising campaign |
| `donate` | Donate XLM to a campaign |
| `withdraw` | Withdraw funds (owner only) |
| `get_campaign` | Get campaign details |
